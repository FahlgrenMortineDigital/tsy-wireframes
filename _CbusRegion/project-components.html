WordPress → Statamic
WordPress Feature Current Plugin / Method Statamic Equivalent Notes
Contact / Lead Forms HubSpot embedded forms HubSpot embed (keep) or Statamic Forms Evaluate if HubSpot CRM integration is needed or if native Statamic forms suffice
Newsletter Signup HubSpot form embed HubSpot embed (keep) or Mailchimp / native form Simple email capture — could be native Statamic form with webhook
Translation / i18n Weglot Not needed English-only site — remove Weglot entirely
Blog / News posts WordPress Posts + Content Views Pro Statamic Collections Create “News” and “Press Releases” collections
Staff / Team directory Custom post type or page Statamic Collection (“Team”) Fields: name, title, organization, headshot, leadership flag
Counties Custom post type / pages Statamic Collection (“Counties”) 11 entries with snapshot data, cities, employers
Industries taxonomy Page sections / accordion content Statamic Collection or Taxonomy 7 industries with descriptions — could be a reusable collection
Page builder / blocks Gutenberg + Columbus Blocks (custom plugin) Statamic Bard / Replicator fieldtype Replicate block types as Bard sets or Replicator sets
PDF downloads Download Monitor Statamic Assets Press release PDFs — store in asset container
Image optimization EWWW Image Optimizer Statamic Glide (built-in) Glide handles responsive images and optimization natively
iFrame embeds Advanced iFrame plugin Bard embed set or raw HTML 19 iframes detected on Economy page alone — audit which are needed

Component 31
Scroll-to-Page Navigation
Used on: Multiple interior pages
Interactive WP Custom JS
Custom Scroll Behavior

When a user scrolls past the end of page content, the browser automatically navigates to a different page. This creates a “single-page” feel across separate WordPress pages.

Recommendation: Evaluate whether to keep or remove. Potential a11y and SEO concerns.
Note: This is a custom JavaScript behavior, not a visual component. It auto-navigates users to the next page in sequence when they scroll past the current page's content. This is unusual and may warrant removal in the redesign for accessibility and SEO reasons.
